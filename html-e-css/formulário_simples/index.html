Exemplo de formulário simples











Neste exemplo, temos um formulário simples com dois campos: "Nome" e "Email". O atributo action define o destino para onde os dados do formulário serão enviados quando o usuário clicar no botão "Enviar". Neste caso, os dados serão enviados para a URL "/submit". O atributo method especifica o método HTTP que será usado para enviar os dados do formulário. Aqui, estamos usando o método "post", que é adequado para enviar dados sensíveis ou grandes quantidades de informações.

Quando o usuário preenche os campos e clica no botão "Enviar", os dados são enviados para o servidor na URL especificada no atributo action, usando o método definido no atributo method. O servidor então processa esses dados conforme necessário.

Resumindo:

  • action="/submit": Define a URL para onde os dados do formulário serão enviados. Normalmente é uma página de backend que processa os dados.
  • method="post": Especifica que os dados do formulário serão enviados usando o método HTTP POST.
  • method="get": Especifica que os dados do formulário serão enviados usando o método HTTP GET.

Esses atributos são essenciais para o funcionamento correto dos formulários HTML e permitem que os desenvolvedores controlem como e para onde os dados do formulário são enviados.

A função TABLE é usada para criar tabelas em HTML. Cada tabela é representada por um elemento 'TABLE', que pode conter linhas 'TR', células de dados 'TD' e células de cabeçalho 'TH'.

A função TR é usada para criar linhas em uma tabela HTML. Cada linha é representada por um elemento 'TR', que pode conter células de dados 'TD' u células de cabeçalho 'TH'.
